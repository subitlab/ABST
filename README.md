# 先进宽带太阳望远镜

先进宽带太阳望远镜（Advanced Broadband Solar Telescope, ABST）是北京大学附属中学天文社和SubIT社团共同设计的开源紫外太阳望远镜项目。本项目的工作波长为工作波长393.4nm（K 谱线），并采用CMOS高速行星相机、幸运成像技术、人工智能等先进技术。
